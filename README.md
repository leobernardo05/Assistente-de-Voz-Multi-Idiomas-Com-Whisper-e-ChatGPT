# 🎙️ Voice Assistant com Whisper + ChatGPT + gTTS

Projeto desenvolvido como laboratório prático no bootcamp Santader GenAI e dados da DIO com o objetivo de criar um Assistente Inteligente por Voz utilizando Python e APIs de IA.

## Sobre o Projeto

Este projeto implementa um assistente virtual que:

1. 🎤 Captura áudio do usuário via navegador (MediaStream API)
2. 🧠 Transcreve a fala utilizando o modelo Whisper
3. 🤖 Envia a pergunta para a API da OpenAI (ChatGPT)
4. 🔊 Converte a resposta em áudio utilizando gTTS
5. ▶️ Reproduz a resposta automaticamente

O sistema foi desenvolvido no Google Colab, integrando Python e JavaScript.
